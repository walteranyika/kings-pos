Local Environment Setup

If you want to use "Intelle Stock Manager" in your local machine but you have no idea how to setup local server, then this section is for you.

Intelle Stock Manager is build by one of the php popular and latest framerwork Laravel. Laravel utilizes Composer to manage its dependencies. So, before using Laravel, you will have to make sure that you have Composer installed on your machine.

Besides you need to create a virtual host in your local machine before using it. You can get an idea about setup from here: http://intelle-hub.com/how-to-create-virtual-host.php
